---

### **Method 2: Local Server (Better)**

#### Windows Users

**Step 1: Install Python**
1. Go to python.org
2. Download Python 3.x
3. Install (check "Add to PATH")

**Step 2: Start Server**
1. Put `commission_site.html` in a folder
2. Open command prompt (cmd)
3. Navigate to folder: `cd C:\Users\YourName\Documents\CommissionSite`
4. Run: `python -m http.server 8000`
5. Open browser: `http://localhost:8000`

**Step 3: Share
1. Find your IP: `ipconfig` (Windows)
2. Share this URL: `http://YOUR_IP:8000/commission_site.html`
3. Others can access from same network

---

#### Mac/Linux Users

**Step 1: Open Terminal**
1. Open Terminal app
2. Navigate to folder with HTML file
3. Run: `python3 -m http.server 8000`

**Step 2: Open in Browser**
1. Go to: `http://localhost:8000`
2. Select `commission_site.html`
3. Website loads

**Step 3: Share**
1. Find your IP: `ifconfig`
2. Share: `http://YOUR_IP:8000/commission_site.html`
3. Others on same network can access

## **🆘 TROUBLESHOOTING INSTALLATION**

### Problem: File won't open
**Solution:**
1. Use Chrome or Firefox browser
2. Make sure it's .html file
3. Check file isn't corrupted
4. Try downloading again

### Problem: Blank page appears
**Solution:**
1. Wait 2-3 seconds (loading)
2. Refresh page (F5)
3. Clear cache (Ctrl+Shift+Delete)
4. Check browser console (F12)

### Problem: Can't login
**Solution:**
1. Try owner code: `13072006aA!`
2. Use STAFF tab for owner
3. Check username/password
4. Try registering new user

### Problem: Can't access from other device
**Solution:**
1. Use web hosting (Netlify, etc.)
2. Not just local file
3. Share the hosted URL
4. Not the file itself

### Problem: Data disappeared
**Solution:**
1. Check if cache was cleared
2. Try opening DevTools → Application → LocalStorage
3. Data should be there
4. Regular backups prevent loss
