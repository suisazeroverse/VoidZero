# 🚀 QUICK START GUIDE — VOID ARTS Commission Studio

## **FIRST TIME SETUP**

### Step 1: Open the Website
1. Open `commission_site.html` in your web browser
2. You'll see the cosmic blackhole animated home page
3. Website is fully functional (no server needed!)

### Step 2: Create Your Account
1. Click **LOGIN** button in top right
2. Click **REGISTER** tab
3. Enter username, email, password
4. Click **CREATE ACCOUNT**
5. Login with your new credentials

### Step 3: Owner Access (Using Owner Code)
1. Click **LOGIN** button
2. Click **STAFF** tab
3. Enter code: `13072006aA!`
4. Click **ACCESS PANEL**
5. You now have full owner access! 👑

## **OWNER (VoidZero) — Full Tutorial**

### 📊 Admin Panel Navigation
After logging in as Owner, click **⚙ PANEL** button (gold color).

### **1. DASHBOARD**
- View system statistics
- See recent orders
- Check queue status
- Quick overview of everything

### **2. COMMISSIONS** 
- Click **+ ADD NEW TYPE** to create commission
- Fill in:
  - Title (e.g., "Demon King Full Body")
  - Category (e.g., "Character Design")
  - Description (what's included)
  - Price in ฿ (Thai Baht)
  - Status: Open / Closed / Waitlist
  - Tags (e.g., character, full body, dark)
  - Delivery Time (e.g., 7-10 days)
- Click **SAVE**
- Commissions now appear on main page!

### **3. ORDERS**
Add orders manually for clients from other sources:
1. Click **+ ADD ORDER**
2. Fill in:
   - Client Username / Name (or Twitter handle)
   - Commission Type (select from list)
   - Description (what they want)
   - Price
   - Status (Pending, In Progress, Completed)
   - Source (Website, Twitter, Discord, etc.)
   - Notes
3. Click **SAVE ORDER**
4. Order appears in queue

### **4. QUEUE**
- View all commission orders in queue
- Change status:
  - **IN PROG** = Currently working on it
  - **DONE** = Finished
  - **REMOVE** = Delete from queue
- Current slot pulses with animation

### **5. GALLERY**
Upload your portfolio images:
1. Enter Title (e.g., "Demon King")
2. Enter Type (e.g., "Full Body")
3. **Upload File** - Click to select image (JPG, PNG, GIF, WEBP)
   - OR enter **Image URL**
4. Click **ADD**
5. Images appear in Gallery page with lightbox viewer

### **6. VIDEOS**
Upload speed paints and tutorials:
1. Enter Title (e.g., "Speed Paint: Demon Overlord")
2. **Upload Video** - Select file (MP4, WebM, MOV, AVI, MKV)
   - OR enter **Video URL** (YouTube, etc.)
3. Click **ADD VIDEO**
4. Videos appear in Videos page with play buttons

### **7. FREEBIES**
Offer free downloads (brushes, textures, references):
1. Enter **File Name** (e.g., "Dark Fantasy Brushes")
2. Enter **Icon** (emoji like 🖌️, 🎨, 📐)
3. Enter **Size** (e.g., 2.4 MB)
4. **Upload File** - Select any file type (ZIP, PDF, BRUSH, etc.)
   - OR enter **Download URL**
5. Click **ADD FILE**
6. Files appear in Freebies page
7. Users must login to download

### **8. REVIEWS**
- View all client reviews
- Click **TOGGLE** to approve/hide reviews
- Click **DEL** to delete reviews
- Only approved reviews show on public page

### **9. CHANNELS**
Create community spaces:
1. Click **+ TEXT CHANNEL**
   - Enter name (e.g., "announcements")
   - Click **CREATE CHANNEL**
2. Click **+ VOICE CHANNEL**
   - Enter name (e.g., "General Voice")
   - Click **CREATE CHANNEL**
3. Users can chat in these channels
4. Click **DEL** to delete channels

### **10. BANS / IP**
Ban misbehaving users:
1. Enter target:
   - Username (to ban account)
   - IP Address (to ban IP)
   - Device ID (to ban device)
2. Enter **Reason** (why they're banned)
3. Select **Type** (Username / IP Address / Device ID)
4. Click **BAN**
5. Banned users cannot access site
6. Click **UNBAN** to remove ban

### **11. PROFANITY FILTER**
Customize blocked words:
1. Enter word to block
2. Click **ADD WORD**
3. All blocked words appear as list
4. Click **✕** to remove a word
5. Filter applies to: chat, reviews, orders, channels

### **12. STAFF CODES** (Owner Only 👑)
Create staff accounts:
1. Enter **New Access Code** (e.g., STAFF_ART_2024)
2. Enter **Rank / Title** (e.g., Artist, Moderator, Colorist)
3. Click **+ ADD CODE**
4. Staff member logs in with:
   - Click LOGIN → STAFF tab
   - Enter the code
5. Click **REVOKE** to remove staff access

### **13. RANKS** (Owner Only 👑)
View all staff ranks:
- Owner rank (VoidZero) - cannot be removed
- All created staff ranks
- Create new rank through Staff Codes

### **14. SETTINGS**
Customize site:
- Commission Status Banner
- Open Slots display
- Artist's Social Media
- Welcome message

## **TROUBLESHOOTING**

### Videos/Images Won't Upload
- File too large? Try smaller file
- Wrong format? Check if format is supported
- Browser storage full? Clear cache and try again

### Chat/Messages Disappearing
- Open browser DevTools (F12)
- Check if LocalStorage is enabled
- Check if you cleared browser data

### Can't Login
- Check username/password spelling
- Account might be banned
- Try creating new account

### Admin Panel Not Showing
- Make sure you logged in as owner/staff
- Use correct access code: `13072006aA!`
- Try logging out and back in
